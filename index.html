📊 Sincronização com Google Sheets (opcional)

Para manter um backup dos seus dados no Google Sheets, crie um Apps Script seguindo os passos abaixo e cole a URL aqui.

Ver instruções de configuração do Apps Script ▸
  1. Abra o Google Sheets e crie uma planilha nova chamada Finanças.
  2. Clique em Extensões → Apps Script.
  3. Apague o código existente e cole o código abaixo.
  4. Clique em Implantar → Nova implantação, tipo App da Web, acesso: Qualquer pessoa.
  5. Copie a URL gerada e cole no campo abaixo.
function doGet(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName('Transações') || SpreadsheetApp.getActiveSpreadsheet().insertSheet('Transações'); const data = sheet.getDataRange().getValues(); return ContentService .createTextOutput(JSON.stringify(data)) .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName('Transações') || SpreadsheetApp.getActiveSpreadsheet().insertSheet('Transações'); const payload = JSON.parse(e.postData.contents); if (payload.action === 'save') { sheet.clearContents(); const header = ['ID','Descrição','Valor','Data','Categoria','Tipo','Fonte']; sheet.appendRow(header); payload.txs.forEach(tx => { sheet.appendRow([tx.id, tx.description, tx.amount, tx.date, tx.category, tx.type, tx.incomeSource||'']); }); return ContentService .createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); } }
💡 Seus dados são salvos automaticamente no localStorage do navegador a cada importação. Não é necessário configurar o Google Sheets para usar o painel — ele é apenas um backup opcional em nuvem.
